## Observations

1. **Degradation curve matches intent** — vessels survive ~30+ min, then steadily decline. By 4h most are gone, 8h nearly total wipe.
2. **SAT crashes hard** — 67% of time at SAT 0 without player input. This is by design (player commands are the primary SAT restoration tool), but relay missions alone can't sustain it.
3. **Wide arc variance** — range [3–230] shows different vessel fates depending on RNG. Lucky relay cycles can extend survival significantly.
4. **14% survive 8h** — some vessels get favorable RNG and persist through the full simulation.

## Tuning Levers

- **SAT decay interval**: `checkSatDecay()` — currently 90–120s between decays
- **Relay SAT restoration**: +1 per CORE completion
- **Phenomena frequency**: global events every 120–300s
- **Phenomena effects**: network_storm (SAT -1), reactor_pulse (SAT +1), relay_launch (SAT +1)
- **Phase damage**: BREACH (30% chance, -1 HP), FAULT (-1 to -2 HP)
- **REBOOT healing**: +1 to +2 HP, +1 to +2 EN, +1 MEM

## How to Run

```bash
node tests/simulate.mjs                    # 10 trials, 8h, 1 vessel
node tests/simulate.mjs --trials 50        # higher confidence
node tests/simulate.mjs --hours 2          # shorter sim
node tests/simulate.mjs --vessels 3        # multi-vessel
```
