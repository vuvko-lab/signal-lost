## Audio Budget for itch.io Bundle

WAV files are large (24-bit 48kHz). For the itch.io zip, we'll need to:
1. **Select a small subset** — pick ~20-30 key sounds, not all 1101
2. **Convert to .ogg or .mp3** — much smaller file size
3. **Categorize chosen files** by game event type

Suggested subset selection:

| Game Event | Source Category | Pick Count |
|------------|----------------|------------|
| New log entry tick | User Interface | 3-4 variations |
| Button click/hover | User Interface | 2-3 |
| Phase transition | Glitch Bits | 3-4 |
| Global phenomenon alert | Bonus (alarms, risers) | 3-4 |
| Damage / integrity loss | Glitch Crackles / Distortions | 2-3 |
| Satellite uplink/downlink | Telemetry | 2-3 |
| BREACH entry | Data | 1-2 |
| Network storm / corruption | Glitch Noises | 2-3 |
| Intercom / broadcast | Intercom | 2-3 |
| Boot sequence | Bonus (ambiences) | 1 |
